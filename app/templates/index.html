Welcome to Flask Supabase Backend

This is a template backend using Flask, Supabase, Redis, and Celery.

Features

  • Flask with Blueprints
  • Supabase (Postgres, Auth, Storage, Realtime)
  • Flask-Smorest for REST API
  • Redis for caching and rate limiting
  • Celery for async tasks
  • Markdown-backed blog system
  • Sentry and OpenTelemetry monitoring

API Endpoints

  • GET /api/v1/health - Health check
  • GET /api/v1/protected - Protected endpoint (requires auth)
  • Blog - Blog listing

Swagger UI - Interactive API documentation
